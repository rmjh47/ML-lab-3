# ML-lab 3
# Retail Sales Exploratory Data Analysis (Lab 3)

## Lab Overview
This repository contains an Exploratory Data Analysis (EDA) of a retail transaction dataset. The goal of this lab is to identify purchasing patterns, top-selling products, and revenue trends to assist in data-driven business decision-making.

## Key Features
- **Data Cleaning:** Handled missing values and removed duplicate records.
- **Visualizations:** Created distributions of sales, revenue trends over time, and payment method preferences using Matplotlib and Seaborn.
- **Statistical Insights:** Analyzed correlations between price, quantity, and total revenue.

## Technologies Used
- Python (Pandas, NumPy)
- Data Visualization (Matplotlib, Seaborn)
